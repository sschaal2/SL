Remarks

This is the generalized inverse dynamics function which computes the NE inverse dynamics for various special cases, as intialized by specific subroutines.

Function Parameters: [in]=input,[out]=output

Parameters
[in]cstate: the current state (pass NULL to use only desired state)
[in,out]lstate: the desired state
[in]endeff: the endeffector parameters
[in]cbase: the position state of the base
[in]obase: the orientational state of the base
[in]ux: the external forces acting on each joint, in world coordinates, e.g., as computed from contact forces
[out]fbase: the force/torque vector of the base in world coordinates
