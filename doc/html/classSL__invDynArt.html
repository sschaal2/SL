Remarks

computes the torques for all joints, and the acceleration of the base

Function Parameters: [in]=input,[out]=output

Parameters
[in]cstate: the current state (pass NULL to use only desired state)
[in,out]lstate: the desired state
[in]endeff: the endeffector parameters
[in,out]cbase: the position state of the base
[in,out]obase: the orientational state of the base
