Remarks

computes the forward dynamics accelerations from the articulate body inertia method

Function Parameters: [in]=input,[out]=output

Parameters
[in,out]state: the state containing th, thd, thdd, and receiving the appropriate u
[in,out]cbase: the position state of the base
[in,out]obase: the orientational state of the base
[in]ux: the external forces acting on each joint, in world coordinates, e.g., as computed from contact forces
[in]endeff: the endeffector parameters
